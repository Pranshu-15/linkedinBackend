## 🚀 Quick Start (Backend)

1. Make sure you have Node.js installed.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your `.env` file based on the root documentation (e.g. `MONGO_URI`, `JWT_SECRET`, `CLOUDINARY_API_KEY`, etc.).
4. Start the development server:
   ```bash
   npm run dev
   ```

This backend is built with **Node.js**, **Express**, **MongoDB (Mongoose)**, and **Socket.io**.
